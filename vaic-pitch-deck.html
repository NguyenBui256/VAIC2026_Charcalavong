Bốn phần ăn khớp thành một vòng lặp

Agent Builder

Cấu hình chuyên gia AI theo phòng ban: KB/RAG riêng, tool calling, tích hợp API, chọn prompt & model.

Orchestrator

Agent điều phối phân rã yêu cầu phức tạp, giao việc qua MCP, fallback về con người khi bí.

Mini-App Builder

Agent tự sinh app: UI + backend + CSDL JSONB riêng, có tầng hiển thị, phát sự kiện ngược lại.

Actions

Lịch cron + trigger sự kiện trong app, kích hoạt agent đúng thời điểm.

VAICTính năng · Agent Builder
01

Agent Builder

Cấu hình chuyên gia AI cho từng phòng ban, với Knowledge Base cô lập: Tín dụng không đọc được tài liệu của Nhân sự.

→KB riêng từng agent: tài liệu chính sách, quy định — truy hồi qua RAG.
→Tool calling: header (auth), input/output schema, Python nhúng tùy chọn.
→Tích hợp API: hệ thống nội bộ, Gmail, Calendar.
→Prompt & model cấu hình được cho từng agent.
Cô lập KB theo phòng ban: yêu cầu của ngành quản chế06
VAICTính năng · Orchestrator
02

Workflow
Orchestrator

Agent điều phối phân rã động một yêu cầu phức tạp thành các task có cấu trúc, rồi giao cho agent chuyên gia.

→Schema task JSON/YAML: task / input / output / expected / criteria.
→Agent phối hợp qua MCP, đồng thời là kho task chia sẻ.
→Xung đột / mơ hồ → fallback về con người kèm trạng thái hiện tại.
→Human-in-the-loop với vòng phản hồi từng bước.
Planner decompose + assign, đúng chuẩn rubric hackathon07
VAICTính năng · Mini-App
03

Mini-App Builder

Từ mô tả + output mong muốn, agent tự sinh một mini-app: UI + backend tự cấp phát + CSDL riêng theo tenant.

→Backend tự động: schema thực thể JSON → bảng JSONB + CRUD endpoint.
→Tầng hiển thị: Public / Need-Auth / Private (whitelist).
→Cô lập theo tenant_id / department_id / owner_id.
→App phát sự kiện ngược vào orchestrator — vòng lặp khép kín.
Điểm khác biệt kiến trúc: agent → app có backend → event08
VAICTính năng · Actions
04

Actions

Kích hoạt agent đúng thời điểm — không cần con người bấm nút.

→Lịch cron: báo cáo hằng ngày, đối soát hằng tuần.
→Trigger sự kiện trong app: "khi một form được gửi".
→Event bus nội bộ: sự kiện mini-app & lịch đều enqueue agent run.
→Là mắt xích khép kín vòng lặp thực thi.
